Question-1: What are the different ways to select an element in the DOM?

We can select DOM elements by using: getElementById, getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent?

InnerHTML gets or sets HTML with tags, innerText gets only visible text, and textContent includes all text, including hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events for child elements using event bubbling, improving performance.

Question-4: What is event bubbling in the DOM?

Event bubbling is a propagation mechanism where an event starts from the target element and moves up through its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

Use document.createElement(tag) to create, parent.appendChild(element) or parent.append(element) to add, and element.remove() to delete.
